## v3版本： node + 后台
采用 webpack + es6 + React + Redux + React-Router + Sass + Iconfont
## 开始
```bash
#先安装node模块
npm install

#开发
npm run start

#生产打包
npm run build
```

## fontend 前端目录
```bash
- config 相关基础配置，v3版本引入了facebook官方的create-app的webpack配置
- mock 假数据
- public 静态文件
- scripts node script打包开发配置
- src 页面主要源文件
    - build 静态重构文件
    - panel 图片
    - scripts javascript 代码
    - styles css 样式代码
    - index.js 入口文件
```
